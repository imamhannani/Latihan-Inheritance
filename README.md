Latihan inheritance ini mengimplementasikan konsep inheritance dalam Java dengan membuat superclass bernama bidangDuaDimensi yang memiliki atribut nama, constructor, serta method info() dan method luas(). Method info() digunakan untuk menampilkan nilai atribut nama, sedangkan method luas() belum didefinisikan sehingga dibuat sebagai abstract method. Selanjutnya dibuat subclass bujurSangkar yang memiliki atribut sisi dan melakukan overriding terhadap method luas() untuk menghitung luas dengan rumus sisi × sisi. Kemudian dibuat subclass Lingkaran yang memiliki atribut jariJari dan mengoverride method luas() menggunakan rumus π × r². Seluruh implementasi dan pengujian program dilakukan pada kelas App.
